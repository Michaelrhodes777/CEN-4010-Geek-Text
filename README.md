## Getting Started

1. Install Visual Studio Code - https://code.visualstudio.com/download

2. Install NodeJS (LTS) - https://nodejs.org/en/download/

3. Install GIT - https://git-scm.com/downloads

4. Run get from CLI in Visual studio code -
 
       npm install -g npm

5. Clone Repository
